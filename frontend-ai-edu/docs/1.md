
1. 디자인 시스템
부산 테마 컬러: 딥블루(#0F3A7D) + 오렌지(#FF6B35) + 중성색
글로벌 CSS 변수: Tailwind 4 기반 커스터마이징
반응형 디자인: 모바일(320px~), 태블릿(768px~), 데스크톱(1024px~) 완벽 지원
우아한 타이포그래피: Google Fonts 통합

수강신청 기능

커뮤니티 게시판 (조회수, 댓글, 좋아요)
게시물 검색 기능

프로젝트 목록 (상태별 필터)
팀 구성 진행률 표시

🛍️ AI 마켓플레이스

카테고리별 필터링

✅ 메인페이지 폰트 가독성 개선 완료!
GlobalBusan 메인페이지의 텍스트 가독성을 대폭 개선했습니다:
개선 사항:
📏 제목 크기 확대: h1 5xl→7xl, h2 3xl→4-5xl로 증가
📝 본문 텍스트 최적화: 기본 텍스트 lg→lg-2xl로 확대
📊 줄 높이 개선: leading-relaxed, leading-tight 적용으로 가독성 향상
🎯 자간(letter-spacing) 추가: tracking-tight/tracking-wide로 텍스트 간격 조정
🎨 색상 대비 강화: text-blue-100→text-blue-50으로 명도 개선
적용 영역:
히어로 섹션 (제목, 부제목)
플랫폼 철학 섹션
5대 모듈 카드
CTA 섹션
푸터
