```

skeleton-load:  콘텐츠 로딩 시 뼈대 UI 표시
```

## 4. 지역별 UX 분화 전략

### 4.1 해운대/수영 (프리미엄)
- **톤앤매너**: 미니멀, 감성, 글로벌
- **폰트**: 영문 혼용 비율 높임
- **이미지**: 아이들 발표 모습, 아이패드, 창작 분위기
- **CTA 문구**: "AI Creative Lab 체험하기"
- **색상 강조**: `--color-sky` 블루 톤 강조

### 4.2 서면/부산진구 (실용)
- **톤앤매너**: 직관적, 결과 지향, 숫자 강조
- **이미지**: 노트북, 업무 장면, 결과 스크린샷
- **CTA 문구**: "AI 부업 시작하기"
- **색상 강조**: `--color-sunrise` 오렌지 에너지

### 4.3 사상/강서 (기업)
- **톤앤매너**: 전문적, 비즈니스, 안정감
- **이미지**: 공장, 오피스, 자동화 대시보드
- **CTA 문구**: "AI 자동화 무료 컨설팅"
- **색상 강조**: `--color-ocean` 딥블루 신뢰감

## 5. 전환 최적화 UX 패턴

### 5.1 무료 설명회 예약 퍼널
```
[히어로 CTA] → [예약 모달 (3 step)]
  Step 1: 이름 + 연락처
  Step 2: 관심 분야 선택 (성인/키즈/기업)
  Step 3: 희망 일정 선택 → [완료]
  → 카카오톡 자동 안내 발송
```

### 5.2 Sticky CTA
- 모바일: 하단 고정 CTA 바 ("무료 설명회 예약" + "문의하기")
- 데스크톱: 스크롤 시 상단 네비에 CTA 버튼 노출

### 5.3 Social Proof 배치
- 모든 주요 CTA 직전에 후기/사례 배치
- "부산 320명이 이미 참여 중" 카운터
- Before→After 사례 카드 (부산 카페 사장님, 학부모 등)

### 5.4 신뢰 요소
- 파트너 로고 바 (제휴 학원/기관)
- 미디어 노출 실적
- 실시간 참여자 수 표시
- 보안 배지 (SSL, 안전결제)

## 7. 접근성 & 성능

### 7.1 접근성 (a11y)
- 색상 대비: WCAG AA 기준 (4.5:1 이상)
- 키보드 네비게이션: Tab 순서, Focus 스타일
- ARIA 레이블: 모든 인터랙티브 요소
- 스크린리더 지원: 의미론적 HTML5 시맨틱 태그
- 다크모드: 시스템 설정 자동 감지 + 수동 토글

### 7.2 성능
- LCP(Largest Contentful Paint): 2.5초 이내
- FID(First Input Delay): 100ms 이내
- CLS(Cumulative Layout Shift): 0.1 이내
- 이미지: WebP 포맷 + lazy loading
- 코드 스플리팅: 라우트 기반 동적 임포트
- CDN: Cloudflare 글로벌 엣지 캐싱
