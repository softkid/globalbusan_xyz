# 부산 AI교육 비즈니스 수익 구조 시뮬레이션

이 사업의 핵심은 단순 “강의비”가 아닙니다.

진짜 돈은:

1. 교육 매출
2. 커뮤니티 매출
3. AI 구축 대행
4. 월 구독형 유지관리
5. 기업 출강
6. 키즈/프리미엄 클래스

에서 나옵니다.

즉:

> “교육 → 구축 → 구독” 구조

가 핵심입니다.
